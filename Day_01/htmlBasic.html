Anatomia de um elemento básico de HTML As tags que é representado pelas "<" e ">" representa o nome do elemento, onde neste caso é o "p". A tag de abertura representa onde o elemento começa e termina na estrutura. Neste caso, é onde o paragrafo começa. A tag de fechamento é a indicação do termino de um elemento, onde neste caso é informado o termino do paragrafo. A sua tag é se estrutura da mesma forma que a tag de abertura, contudo, possui um slash "/" antes do nome. A falha no fechamento dos elementos pela suas tags poderá ocasionar problemas e erros em seu código. O que está entre as tags é o conteudo do elemento, neste caso é um paragrafo de texto. O elemento HTML indica todo a sua estrutura de tags, abertura e fechamento e elemento.

Os elementos possuem a opção de se terem atributos associados a eles. Estes atributos podem atribuir informações extras ao elemento onde não é necessário informar ou ilustrar ao usuario final. Neste caso, o atributo é o "class", que neste caso é o nome do atributo, e seu valor "editor-note", que representa o valor deste atributo. O atributo 'class' permite que você atribua um identificador não unico que irá dar informações e estilos e dentre outras coisas ao seu elemento. O atributo com um set de valor precisa sempre ter: Um espaço entre o o nome do elemento e/ou atributo anterior e o atributo atual; O nome do atributo terá que ser seguido com um igual "="; O valor do atributo sempre terá que ser seguidor por aspas unicas => '' Nota: É possível sim aplicar valores de atributos sem as aspas contanto que ela não tenha caracteres especiais e/ou espaços ASCII. Contudo, para facilitar a leitura é sempre recomendado a aplicação delas.
