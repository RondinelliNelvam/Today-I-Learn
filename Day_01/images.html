A tag img tem como função fixar uma imagem dentro de algum reposítorio ou link para a página da web. Isso é feito via o atributo src onde seu valor é o path(caminho) para a imagem. O atributo alt é utilizada para informar o texto(nome) da imagem para os usuarios que não conseguiram visualizar a imagem. Isso pode ocorrer por varios motivos como o caminho do texto está incorreto.
