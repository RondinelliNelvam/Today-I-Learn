Utilizando o exemplo feito anteriormente acima como exemplo, possuímos: !DOCTYPE html — o doctype. É a parte inicial obrigatória do documento. Antigamente, doctypes eram criados para agir como links para um conjunto de regras que a página HTML tinha que seguir para ser considerada um bom HTML, o que poderia significar uma verificação automática de erros e outras coisas úteis. Contudo, atualmente não são mais utilizados para este fim e são basicamente necessários apenas para garantir que o documento se comporte corretamente. html — o elemento html. Esse elemento envolve todo o conteúdo da página e às vezes é conhecido como o elemento raiz. head — o elemento head. Esse elemento age como um recipiente de tudo o que você deseja incluir em uma página HTML que não é o conteúdo que você quer mostrar para quem vê sua página. Isso inclui coisas como palavras-chave e uma descrição que você quer que apareça nos resultados de busca, CSS para dar estilo ao conteúdo, declarações de conjuntos de caracteres e etc. meta — Esse elemento define o conjunto de caracteres que seu documento deverá usar. Neste caso é o UTF-8, que inclui praticamente todos os caracteres da grande maioria dos idiomas escritos. Essencialmente, agora ele pode manipular qualquer conteúdo textual que você possa colocar. Não há razão para não definir isso e assim pode ajudar a evitar alguns problemas no futuro. title — Esse elemento define o título da sua página, que é o título que aparece na guia do navegador onde sua página é carregada. Ele também é usado para descrever a página quando você a adiciona aos favoritos. body — Esse elemento contém todo o conteúdo que você quer mostrar ao público que visita sua página, seja texto, imagens, vídeos, jogos, faixas de áudio reproduzíveis ou qualquer outra coisa.
