But, this is another list, but an Ordered One

  1. technologists
  2. thinkers
  3. builders
Listas no HTML são feitas pela tag ul ou ol. Onde a tag ul significa lista desordenada, ou unordered list, onde a ordem dos itens não importa. Enquanto na tag ol é a lista ordenada, ou ordered list onde a ordem da lista importa. Em ambas as tags, cada item da list tem que ser aplicada dentro de uma tag "li"
